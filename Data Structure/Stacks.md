## Overview

Stack is a linear data structure that follows the First In Last Out (FILO) ordering.

---

## Time Complexity

| Operation | Runtime | Example                     |
| --------- | ------- | --------------------------- |
| Push      | O(1)    | `stack.append(value)`       |
| Pop       | O(1)    | `stack.pop()`               |
| Peek      | O(1)    | `stack[-1]`                 |
| isEmpy    | O(1)    | `isEmpty = len(stack) == 0` |
| Size      | O(1)    | `len(stack)`                |
